# Movie Sentiment Frontend

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Features

- 🏠 **Home Page**: Search for movies by name or IMDb link
- 📊 **Results Dashboard**: Visualize sentiment analysis with charts
- 💬 **Review Analysis**: Detailed review analysis with keyword highlighting
- 🎨 **Beautiful UI**: Modern glassmorphism design with Tailwind CSS
- 📈 **Real-time Analytics**: Sentiment percentages, emotion breakdown, keyword extraction

## Environment

Make sure the FastAPI backend is running on `http://localhost:8000` before starting the frontend.
